# Changing state in [React](https://reactjs.org/)

``` js
const { count } = this.state;
this.setState({
  count: count + 1
});
```

---

# Changing state in [Svelte](https://svelte.dev/)

``` js
let count = 0;

// later...
count += 1;
```

# Assign Operator

``` js
let a = 10;
let b = a + 1;
// b === 11;

a = 20;
// b === 11;
```

---

# Destiny Operator

``` js
let a = 10;
$:  b = a + 1;
// b === 11;

a = 20;
// b === 21;
```
